# 📚 AI Study Planner

AI Study Planner is a smart web application designed to help students manage their study time efficiently. The application creates a personalized study plan based on subject priorities and available study hours.

## 🚀 Features

* Add subjects and study hours
* Set priority levels (High, Medium, Low)
* Generate a smart study schedule
* Dynamic user interface
* Local Storage support
* Responsive design

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

## 📂 Project Structure

AI-Study-Planner/

├── index.html

├── style.css

├── script.js

└── README.md

## ⚡ How to Use

1. Enter subject name
2. Select priority level
3. Add study hours
4. Click "Add Subject"
5. Click "Generate Study Plan"
